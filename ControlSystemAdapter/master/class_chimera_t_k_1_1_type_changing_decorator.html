Detailed Description

template<class T, class IMPL_T>
class ChimeraTK::TypeChangingDecorator< T, IMPL_T >

Strictly this is not a real decorator.

It provides an NDRegisterAccessor of a different template type and accesses the original one as "backend" under the hood.

This is the base class which has pure virtual convertAndCopyFromImpl and convertAndCopyToImpl functions. Apart from this it implements all decorating functions which just pass through to the target.

This class is not thread-safe and should only be used from a single thread.
