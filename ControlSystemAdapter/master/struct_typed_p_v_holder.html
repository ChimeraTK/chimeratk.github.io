◆ dataTypeConstant

template<class DataType >
ChimeraTK::ProcessArray<DataType>::SharedPtr TypedPVHolder< DataType >::dataTypeConstant

The "data type constant" is a value that depends on the data type.

It is intended as 'magic' constant which can be read out to test reading because the value it knows.

The value is:

  • sizeof(type) for unsigned integer types
  • -sizeof(type) for signed integer types
  • 1./sizeof(type) for floating point types
