Detailed Description

A helper class to implement the communication to the XPORT GPIO pins.
You can configure whether a pin is active high or active low, input or output and, for the output pins, whether it is active or not.

def xport_gpio.XPortGpio.set_active_levels (   self,
  stateword,
  pinmask = 7 
) 
stateword is a bit field with the active level configuration for each pin.
1 is active low, 0 is active high.
The pin mask allows to leave pins with unknown level untouched. With the default 
7 all three pin states are applied.
