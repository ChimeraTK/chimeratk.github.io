Detailed Description

This class implements a minimal telnet client for use in programmes.

It treats all incomming telnet sequences correctly (filters them from the data stream and answers negotiation requests of the server.) It is a minimal implementation which refuses all requests from the server, does not enable any telnet options and never starts negotiations with the server.

It is intended for synchronous, automated use, which is implemented like this:

  1. Upon connection, the client reads until it receives a prompt signature from the server
  2. The client sends a command and waits for the response, whose end again is recognised by the prompt.
