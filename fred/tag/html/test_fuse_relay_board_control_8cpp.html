◆ getTestFilter()

std::string getTestFilter ( std::string const &  protocolVersion)

Returns test case filter string which selects all required testsuites required to test the given protocolVersion.

Throws a runtime error if protocolVersion is unsupported.

Example usage: getTestFilter("2.0") -> "--run_test=v2.0" getTestFilter("2.1") -> "--run_test=v2.0,v2.1" getTestFilter("2.2") -> "--run_test=v2.0,v2.1,v2.2"

Parameters
protocolVersionversion string in the form major.minor. Function throws a runtime error if protocolVersion not in {"2.0", "2.1", "2.2"}
Returns
string in the form "--run_test=v2.0,v2.1". This string can be passed to the boost runner to pick the required test suites.
