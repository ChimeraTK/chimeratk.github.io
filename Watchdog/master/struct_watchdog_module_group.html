LogFileModule WatchdogModuleGroup::logFile {this, ".", "Logging module reading the watchdog logfile", "/Trigger/tick", "/logging/logFile"}

This module is used to read the watchdog log file, which includes messages from the watchdog process and all other processes controlled by the watchdog.

This allows to show all watchdog related messages via the tail (status/logTailExternal). The LogFileModule is used differently for all the processes, since in case of the processes the messages produced by the started processes are read and shown in the corresponding tail (status/logTailExternal). Thus in that case the postfix 'External' makes more sense. Here the postfix 'External' can be a bit misleading, because the tail (status/logTailExternal) includes only messages produced by the watchdog itself.

Remarks
Here the log file setting from the logging module is used directly by moving the logFile PV to the logging module location.
