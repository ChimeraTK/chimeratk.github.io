◆ TriggerType

enum ChimeraTK::VariableNetwork::TriggerType
strong

Define trigger types.

The trigger decides when values are fed into the network and distributed to the consumers.

Enumerator
feeder 

The feeder has an UpdateMode::push and thus decides when new values are fed.

pollingConsumer 

If there is exacly one consumer with UpdateMode::poll, it will trigger the feeding.

external 

another variable network can trigger the feeding of this network

none 

no trigger has yet been selected
