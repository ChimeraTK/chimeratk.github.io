Detailed Description

template<typename UserType>
class ChimeraTK::ConsumingFanOut< UserType >

FanOut implementation which acts as a read-only (i.e.

consuming) NDRegisterAccessor. The values read through this accessor will be obtained from the given feeding implementation and distributed to any number of slaves.
