static void ChimeraTK::Profiler::registerThread ( const std::string &  name)
inlinestatic

Register a thread in the profiler.

This function must be called in each thread before calling startMeasurement() and stopMeasurement() in the same thread. The function must not be called twice in the same thread. The call to this function implicitly triggers starting the time measurement (see startMeasurement())
