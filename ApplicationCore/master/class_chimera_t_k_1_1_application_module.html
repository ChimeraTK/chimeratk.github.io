Constructor: Create ApplicationModule by the given name with the given description and register it with its owner.

The hierarchy will be modified according to the hierarchyModifier (when VirtualModules are created e.g. in findTag()). The specified list of tags will be added to all elements directly or indirectly owned by this instance.

Note: ApplicationModules may only be owned by ModuleGroups or Applications.

void ChimeraTK::ApplicationModule::decrementDataFaultCounter ( )
overridevirtual

Decrement the fault counter and set the data validity flag to ok if the counter has reached 0.

This function will be called by all input accessors when receiving the an ok update if the previous update was faulty. The caller of this function must ensure that calles to this function are paired to a previous call to incrementDataFaultCounter().
