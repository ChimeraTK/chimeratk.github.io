Detailed Description

template<typename Derived>
class ChimeraTK::InversionOfControlAccessor< Derived >

Adds features required for inversion of control to an accessor.

This is needed for both the ArrayAccessor and the ScalarAccessor classes, thus it uses a CRTP.
