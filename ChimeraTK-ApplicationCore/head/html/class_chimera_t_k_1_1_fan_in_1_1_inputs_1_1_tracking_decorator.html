Detailed Description

template<push_input AccessorType>
template<user_type U>
class ChimeraTK::FanIn< AccessorType >::Inputs::TrackingDecorator< U >

Helper decorator which keeps track of the last update received by the FanIn.

This is needed because the ReadAnyGroup is created by user code and we do not want to require the user code to pass us the last changed TransferElementID.
