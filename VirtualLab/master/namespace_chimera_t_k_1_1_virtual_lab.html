typedef int64_t ChimeraTK::VirtualLab::VirtualTime

Type and units for virtual time.

Important note: since 64 bit integer is used as a type, attention has to be paid when converting from double, since the precision might not be sufficient. Example: "100*days + 1*picoseconds" will work as expected, while "100*days + 1.*picoseconds" (with additional decimal point) will be equal 100*days, since double precision is not sufficient to resolve the difference.
