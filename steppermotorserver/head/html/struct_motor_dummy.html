Detailed Description

MotorDummy - A dummy application module to employ the MotorControlerDummy functionality of the MotorDriverCard library.

Note: This is a workaround to handle the dummy provided by the MotorDriverCard library, which requires specific function calls not supported by the real MotorControler.
