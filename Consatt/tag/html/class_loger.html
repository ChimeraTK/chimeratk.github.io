◆ NO_LOGGING

const Loger::LogLevel Loger::NO_LOGGING
static

Available logging levels created as a constant static fields.

They are used to mark logging level of messages. The MAX_DETAIL level is highest and the NO_LOGGING is lowest.

  • When logging level of Loger class object is set to MAX_DETAIL (highest), all messages are logged (except of ones marked as NO_LOGGING).
  • When logging level of Loger class object is set to DETAIL, messages marked as ERROR, WARNING, INFO, DETAIL are logged.
  • When logging level of Loger class object is set to INFO, messages marked as ERROR, WARNING, INFO are logged.
  • When logging level of Loger class object is set to WARNING, messages marked as ERROR, WARNING are logged.
  • When logging level of Loger class object is set to ERROR, messages marked as ERROR are logged.
  • When logging level of Loger class object is set to NO_LOGGING, no messages are logged (even marked as NO_LOGGING)
