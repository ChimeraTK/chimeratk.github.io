Binary packages
Ubuntu 20.04 packages are available in the DESY DOOCS repository, which can be added to your system like this: 
wget -O - https://doocs-web.desy.de/pub/doocs/DOOCS-key.gpg.asc | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/doocs-keyring.gpg
sudo sh -c 'echo "deb https://doocs-web.desy.de/pub/doocs  jammy main" > /etc/apt/sources.list.d/doocs.list'
