Detailed Description

template<typename UserType, typename TargetUserType = UserType>
class ChimeraTK::NDRegisterAccessorDecorator< UserType, TargetUserType >

Base class for decorators of the NDRegisterAccessor.

This is basically an empty decorator, so implementations can easily extend by overriding only single functions (and usually calling the implementations of this class inside the overrides).
