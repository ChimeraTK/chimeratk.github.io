◆ _canonicalInterrupts

std::map<RegisterPath, std::vector<size_t> > ChimeraTK::NumericAddressedRegisterCatalogue::_canonicalInterrupts
protected

A canonical interrupt path consists of an exclamation mark, followed by a numeric interrupt and a colon separated list of hierarchical sub interrupts.

For each interrupt with sub levels there is always a canonical interrupt for all higher levels.

Example: For the canonical interrupt !3:5:9 there are is an interrupt !3:5 and the primary interrupt !3.
