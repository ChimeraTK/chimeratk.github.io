Detailed Description

Abstract class.

It is used to manage database for particular cavity tests. The abstract functions that has to be overriden are: virtual std::vector<amtfDbMod::testResults*> processTestResults(const pq xx::result& result) - a protected method that is processing the test results constructor, where you specify viewName and tableName testResults nested class, where you have to override getScalars() and getVectors() methods.
