Detailed Description

template<class RAW_DATA_TYPE, class COOKED_DATA_TYPE>
class RegisterTypeAbstractorRawImpl< RAW_DATA_TYPE, COOKED_DATA_TYPE >

This is a double templated accessor abstractor with raw and cooked data type.

It is based upon the single-typed RegisterTypeAbstractorImpl with the raw type as user type. For those functions which do cooked values, it does the corresponding conversion.
