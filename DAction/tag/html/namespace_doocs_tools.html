Convenience function to check that the history filter setting have a certain value.

The default parameters are for the advanced mode as they arrive in init.

Note that this function is templated because d_fct does not have a get_history function. You have to put in a D_int, D_float etc.

Convenience function to set the engineering units of a doocs property (or to be correct of its history).

Note that the order of the parameters is different to D_hist::egu() to be able to use default parameters. Also note that this function is templated because d_fct does not have a get_history function. You have to put in a D_int, D_float etc.

Parameters
propertyThe Doocs property
eguTextThe enineering unit (which is just a string)
linLogLinar or log scale of the y axis (see D_hist)
startMinimum of the y axis (see D_hist)
endMaximum of the y axis (see D_hist)
