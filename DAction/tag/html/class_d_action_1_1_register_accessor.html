Detailed Description

The RegisterAccessor class.

This class has been imported from DeviceAccess to simplify the transition of downstream projects so that ActionDeviceAccess can still be used.

Instead of device.getRegisterAccessor(...) just use DAction::RegisterAccessor::create(device, ...)

◆ _convertingAccessorHandlers

ChimeraTK::TemplateUserTypeMap<AccessorHandler> DAction::RegisterAccessor::_convertingAccessorHandlers
mutableprotected

The converting accessors used under the hood.

They are not initialised in the constructor but only when first used to safe memory. Usually you will not use read or write of all user data types. Thus the variable is mutable to allow initialisation in the const read and write function.
