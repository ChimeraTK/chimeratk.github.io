Detailed Description

template<bool asyncOnGet = false>
class D_StdStringExtended< asyncOnGet >

A class that implements the component pattern design in DOOCS properties for D_string only.

It is a variation of D_fctExtended class, but internally it works on std::string instead of const char *. This allows to use Actions created against std::string instead of const char *, which solves some issues with boost::any usage.
