Detailed Description

template<typename T>
class ActionDeviceAccess< T >

Class fulfilling the interface of ActionBase class.

It allows to use RegisterAccessor to access data from ChimeraTK::Device instances. Please note, that although the RegisterAccessor is deprecated, it is still used in there and it will be changed in some newer version of the library.
