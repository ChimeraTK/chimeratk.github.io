Detailed Description

A class to controll stepper motors using the DFMC-MD22 card.

This class represents the base functionality of one motor.

Note: Technically this controler class addresses registers of the TMC 429 motor controler chip (which can handle up to 3 motors) and the TMC 260 motor driver, which is indiviual for each motor. On the software side these implementation details are not important. Think of this class as a virtual controler with integrated driver.

This purely virtual class defines the interface.
