The ADD_VARIABLE macro allows to add a getter and a setter function for a variable with the correct input and output masks.

The user specifies the variable name and the first and last bit of the sub word.

ADD_VARIABLE( Voltage, 12, 15 )

expands to 

void setVoltage(unsigned int);
unsigned int getVoltage();

including the implementation for a 4 bit wide sub word (12 through 15, both bits included, mask 0xF000)

◆ INPUT_MASK

#define INPUT_MASK (   FIRST_BIT,
  LAST_BIT 
)    static_cast<unsigned int>((1UL << (LAST_BIT - FIRST_BIT + 1)) - 1)

Generate an input mask from the position of the first and the last bit of the mask.

The input mask is the mask applied to an input word which is going to be written. As these are always the last bits of the input, the mask is only dependent on the width of the sub word.
Example: bits 12 to 15 describe a 4 bit wide sub word with an input mask of 0x0000000F. This macro is mainly for internal usage in the ADD_VARIABLE() macro.

Attention
There is no range check for the bits. The valid bit range is [0:31] for both first and last bit.

Generate an outpu mask from the position of the first and the last bit of the mask.

The output mask is the mask applied to the combined word.
Example: bits 12 to 15 describe a 4 bit wide sub word with an output mask of 0x0000F000. This macro is mainly for internal usage in the ADD_VARIABLE() macro.

Attention
There is no range check for the bits. The valid bit range is [0:31] for both first and last bit.
