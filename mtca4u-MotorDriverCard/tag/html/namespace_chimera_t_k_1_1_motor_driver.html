◆ CalibrationMode

enum class ChimeraTK::MotorDriver::CalibrationMode
strong

Enum type describing how the system is calibrated:

0 - NONE: No calibration has been determined or it has been lost.
1 - SIMPLE: The motor position has been defined by setActualPosition().
No valid position for the end switches can be determined. Useful if driving in full range between the end switches is possible.
2 - FULL: The motor has been calibrated by calling calibrate(). The end switch position has been determined.

◆ ExitStatus

enum class ChimeraTK::MotorDriver::ExitStatus
strong

Enum describing the exit status of setting member functions.

The member functions for setting parameters to the StepperMotor object return this to provide user feedback in case an input can not be applied due to the system's current state or an invalid value.
