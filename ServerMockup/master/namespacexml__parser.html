◆ getRootDir()

std::string xml_parser::getRootDir ( std::string  inputFile)

Open the xml file and retrive the root directory.

This might be used to define the server name. The expected structure in the xml is:

<application xmlns="https://github.com/ChimeraTK/ApplicationCore" name="WatchdogServer">
....
</application>

In this example the returned name is WatchdogServer.

Parameters
inputFileInput xml file produced with the srervers xml generator
Returns
The application name.
Exceptions
xmlpp::exceptionIn case no application element is found.
