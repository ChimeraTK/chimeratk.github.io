Detailed Description

This module holds all control system variables read from the input xml file.

Remarks
In principle no TimerModule is needed here. One could simply use a trigger variable in the control system, which will never be changed. Thus the main loop of the XMLModule would be blocked forever.
Can not inherit from TemplateGroup since both inherit from ctk::Module which is not virtual inherited. Else one could get rid of defining addElement here and in TemplateGroup.
