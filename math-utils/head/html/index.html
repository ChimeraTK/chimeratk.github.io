math-utils

Support library for recurring calculations

Math-utils is a header-only library that aims to provide developers with default implementations for recurring statistical calculations, such as mean, standard deviation, RMS on different bodies of values.

There are classes for statistic calculations on real values as well as angular values.

MathUtils::SimpleStatistics

Provides statistics for an unlimited number of real values

MathUtils::RingbufferStatistics

Provides a simple moving average implementation of the past n values. Similiar to SimpleStatistics, but has a limited number of samples, where the newest added value will replace the oldest value.

MathUtils::CircularStatistics

Calculation of mean direction of angles, as well as standard deviation of this circular quantities.

Usage

For use in a CMake file, it is possible to use the usual FIND_PACKAGE mechansim:

find_package(math-utils REQUIRED)
...
target_link_library(MyProject math-utils)

In the code, either use the helper file math-utils.h to include everything or opt on using the indivdual header files.

#include <math-utils.h>
