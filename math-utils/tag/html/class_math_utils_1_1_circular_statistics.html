Detailed Description

template<class T>
class MathUtils::CircularStatistics< T >

This class calculates the standard deviation of circular quantities as well as mean direction and the variance.

The unit of the n input , and output values of mean(), meanDirection() and circularStandardDeviation() is degrees.

There are two different implementations to calculate the mean direction , mean() and meanDirection(), see their individual documentation for details.

But every implementation has to calculate these two sums of sine and cosine using the most of the CPU time. At least the function calcSumOfSinCos() is called, if it is needed. But you can call this function yourself have full control over CPU time usage.

The data can be added either on construction, completely replaced using one of the the assign() methods or extended using add().

If add() is called direct after assign(), it will call calcSumOfSinCos() first, this will cost CPU time. It is recommended not to use assign() and add() together.

Create an object using C++ LegacyInputIterators.

This lets you create circularStatistics objects from STL container iterators (vectors, lists etc..).

Note
The data from the given range will be copied into the object.
Parameters
firststart iterator
lastend iterator

Example:

std::list<int> l{1, 2, 3};
CircularStatistics<double> s1{l.begin() + 1, l.end()}; // {2, 3}
std::vector<int> v = {1, 2, 3}
CircularStatistics<float> s2{v.begin(), l.end()}; // {1, 2, 3}
