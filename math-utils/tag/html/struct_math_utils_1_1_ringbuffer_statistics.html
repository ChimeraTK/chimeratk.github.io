Detailed Description

template<typename T>
struct MathUtils::RingbufferStatistics< T >

This class provides a simple wrapper around SimpleStatistics class providing convenience for implementing statistics over the last size values.

The internal list will grow until it reaches the configured size and will then begin to overwrite slots in a FIFO manner.

For supported calculations, see SimpleStatistics itself.
