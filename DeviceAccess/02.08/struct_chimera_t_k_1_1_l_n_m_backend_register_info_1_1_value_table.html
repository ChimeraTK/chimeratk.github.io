Detailed Description

template<typename T>
struct ChimeraTK::LNMBackendRegisterInfo::ValueTable< T >

Hold values of CONSTANT or VARIABLE types in a type-dependent table.

Only the entry matching the valueType is actually valid. Note: We cannot directly put the std::vector in a TemplateUserTypeMap, since it has more than one template arguments (with defaults).
