DeviceInfoMap::DeviceInfo const & ChimeraTK::DMapFilesParser::getdMapFileElem ( const std::string &  devName)

Get the dRegisterInfo from the device name.

Attention
The reference is only valid as long as the dmapFilesParser object is in scope and no new parse action has been performed. In practice this should not be a limitation. Just use 
dmapFile::dRegisterInfo
myDeviceInfo = myDmapFilesParser->getdMapFileElem(deviceName); //creates a
copy
instead of 
dmapFile::dRegisterInfo const & myDeviceInfo =
myDmapFilesParser->getdMapFileElem(deviceName); //does not create a
copy
