virtual std::string ExampleBackend::readDeviceInfo ( )
inlinevirtual

Return a device information string containing hardware details like the firmware version number or the slot number used by the board.

The format and contained information of this string is completely backend implementation dependent, so the string may only be printed to the user as an informational output. Do not try to parse this string or extract information from it programmatically.

◆ backendRegisterer

ExampleBackend::BackendRegisterer ExampleBackend::backendRegisterer
static

The one static instance of the registerer.

Currently we keep it public so there is an object which can be used in the client code. This trick is needed to force the library to be loaded as long as the loading mechanism is not implemented into the dmap file (see DeviceClient.cpp how to do it).
