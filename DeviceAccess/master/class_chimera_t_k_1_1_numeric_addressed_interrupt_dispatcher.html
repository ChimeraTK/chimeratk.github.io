Detailed Description

The NumericAddressedInterruptDispatcher has two main functionalities:

  • It calls functions for all asynchronous accessors associated with one interrupt
  • It serves as a subscription manager

This is done in a single class because the container with the fluctuating number of subscribed variables is not thread safe. This class has implements a lock so dispatching an interrupt is safe against concurrent subscriptions/unsibscriptions.
