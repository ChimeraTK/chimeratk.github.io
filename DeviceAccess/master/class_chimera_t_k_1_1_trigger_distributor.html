Detailed Description

Distribute a void type interrupt signal (trigger) to three possible consumers:

  • InterruptControllerHandler
  • TriggeredPollDistributor
  • VariableDistributor<nullptr_t>

Common implementation for getting a TriggeredPollDistributor or a VariableDistributor<nullptr_t> to avoid code duplication.

It only works for those two template types. The implementation and the code instantiations are in the .cc file to avoid circular header inclusion.
