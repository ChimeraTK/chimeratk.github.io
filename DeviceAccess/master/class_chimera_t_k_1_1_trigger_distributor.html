Detailed Description

template<typename BackendSpecificDataType>
class ChimeraTK::TriggerDistributor< BackendSpecificDataType >

Distribute a typed interrupt signal (trigger) to three possible consumers:

  • InterruptControllerHandler
  • TriggeredPollDistributor
  • VariableDistributor<BackendSpecificDataType>

Common implementation for getting a TriggeredPollDistributor or a VariableDistributor<nullptr_t> to avoid code duplication.
