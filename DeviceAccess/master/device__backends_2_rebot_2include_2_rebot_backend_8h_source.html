30  // A helper class that contains a mutex and a flag.
31  // The idea is to put it into a shared pointer and hand it to a thread which is
32  // sleeping a long time. You can then detach the thread, tell it to finish and
33  // continue with the constructor without having to wait for the tread to wake up
34  // and finish before you can join it. The thread locks the mutex and checks if
35  // it should finish when it wakes up, which it can do because the mutex and flag
36  // still exist thanks to the shared pointer.
37  struct ThreadInformerMutex {
38  std::mutex mutex;
39  bool quitThread;
40  ThreadInformerMutex() : quitThread(false) {}
41  };
