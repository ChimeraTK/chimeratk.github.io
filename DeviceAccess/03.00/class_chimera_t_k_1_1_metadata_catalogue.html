Detailed Description

Container for backend metadata.

Metadata is additional information provided by the backend. It can come e.g. from the map file (if existent) and/or may provide e.g. information about the firmware version of the device etc. Which metadata is provided is backend specific, so applications should not rely on the presence of certain metadata.

The metadata is a list of key-value pairs with string data types for both key and value.
